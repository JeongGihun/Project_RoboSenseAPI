Architecture
시스템 아키텍처

Nginx 리버스 프록시 뒤에 다중 FastAPI 인스턴스가 비동기로 동작하며, Redis 캐싱과 PostgreSQL을 통해 높은 처리량과 낮은 지연 시간을 달성합니다.

Tech Stack
기술 스택

FastAPI + Uvicorn

비동기 ASGI 서버, 타입 힌트 기반 자동 문서화

PostgreSQL 16

asyncpg 드라이버, JSONB 센서 데이터, 복합 인덱스

Redis 7

Cache-Aside 패턴, TTL 기반 자동 만료, 캐시 워밍

C++ 17 + pybind11

센서 직렬화, 이동평균 필터링, null 비율 계산

Nginx

리버스 프록시, 로드 밸런싱, GZip 압축

Docker Compose

멀티 컨테이너 오케스트레이션, 환경 일관성

AWS EC2

실제 배포 환경에서 TPS 부하 테스트

Locust

점진적 부하 테스트 (100 → 5,000 동시 접속)
